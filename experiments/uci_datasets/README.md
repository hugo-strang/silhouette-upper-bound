# UCI Datasets

This directory contains experiments on UCI datasets. These datasets are publicly available and can be downloaded from the links below.

| Dataset | url | 
| --- | --- | 
| Ceramic | https://archive.ics.uci.edu/dataset/583/chemical+composition+of+ceramic+samples | 
| Customers | https://archive.ics.uci.edu/dataset/292/wholesale+customers | 
| Dermatology | https://archive.ics.uci.edu/dataset/33/dermatology | 
| Heart Statlog | https://archive.ics.uci.edu/dataset/145/statlog+heart | 
| Optdigits | https://archive.ics.uci.edu/dataset/80/optical+recognition+of+handwritten+digits | 
| Rna | https://archive.ics.uci.edu/dataset/401/gene+expression+cancer+rna+seq | 
| Wdbc | https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic | 
| Wine | https://archive.ics.uci.edu/dataset/109/wine | 

## Setup & Generation

The notebook `uci_data.ipynb` should be executed first. The script assumes that the datasets exist in a datasets/ folder.
